# AI 全栈极速黑客松 — 开发时间线

> 剩余时间：3 小时 40 分钟（220 分钟）
> 已完成：需求对齐、技术选型、项目骨架（基础目录结构+依赖安装）

---

## 技术栈（已确认）

| 层 | 选择 |
|---|---|
| 前端 | React 19 + TypeScript + Vite + shadcn/ui + Tailwind CSS + zustand |
| 可视化 | ECharts（力导向图） |
| 后端 | FastAPI + Python |
| PDF 解析 | PyMuPDF + pymupdf4llm + 自定义过滤（页眉/图表/表格标题保留） |
| Embedding | bge-small-zh-v1.5（本地，sentence-transformers） |
| 向量库 | ChromaDB |
| LLM | MiMo 2.5 Pro（文本主模型）+ MiMo Omni（多模态加分项） |
| 架构 | 多 Agent 共享内存（Python 模块化） |
| 部署 | Docker → 魔搭创空间 |

---

## 5 Agent 架构

```
ParserAgent    → PDF/MD/TXT 解析，章节检测，表格标题提取
ExtractorAgent → 逐章 LLM 知识点+关系提取，三级递进跨章关系
IntegratorAgent → Embedding 粗筛 + LLM 精确判断，整合决策
RAGAgent       → 分块→embedding→ChromaDB→检索→带引用生成
ChatAgent      → 意图识别→调用其他 Agent API→实时更新图谱
```
